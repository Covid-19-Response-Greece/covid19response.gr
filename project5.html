Project Data Analytics

Συλλογη & Οπτικοποιηση των δεδομενων εξαπλωσης του COVID-19 στην Ελλαδα Τελευταια ενημερωση: 22/4/20

Στοχος του Project


  • Συγκέντρωση δεδομένων εξάπλωσης του COVID-19 στην Ελλάδα από ποικίλες πηγές με σκοπό την κατασκευή μιας οργανωμένης και ενημερωμένης βάσης η οποία διατίθεται για χρήση σε ερευνητές και αναλυτές.
  • Ανάπτυξη ενός δημοσίως διαθέσιμου API για παροχή real-time δεδομένων.
  • Κατασκευή οπτικοποιήσεων για την αποτύπωση της εξάπλωσης του COVID-19 στην Ελλάδα σε πραγματικό χρόνο.
  • Παραγωγή λογισμικού ανοιχτού και προσβάσιμου σε επίσημους φορείς και υπηρεσίες του κράτους και του συστήματος υγειονομικής παρακολούθησης χωρίς χρέωση κάτω από άδεια χρήσης European Union Public Licence (EUPL-1.2).
  • Συντονισμός και παροχή τεχνικής υποστήριξης προς τους ενδιαφερόμενους φορείς για την ανάπτυξη και προσαρμογή λογισμικού στα πλαίσια της παρακολούθησης και διαχείρισης της πανδημίας.

Λυση 1: Συγκεντρωση & Οργανωση δεδομενων απο ποικιλες πηγες

  1. 1. Περιγραφη
  2. 2. Συντελεστες
  3. 3. Συνδεσμοι

1. Περιγραφη


Μια από τις πιο χρονοβόρες διαδικασίες σε μια ανάλυση είναι η εύρεση αξιόπιστων δεδομένων και η μετατροπή τους σε μια μορφή εύκολα διαχειρίσιμη.

Ο στόχος μας είναι η συγκέντρωση δεδομένων από διάφορες πηγές και η οργάνωση τους, με κίνητρο την ενθάρρυνση και διευκόλυνση της έρευνας σχετικά με τον COVID-19. Παρέχουμε μια οργανωμένη και ενημερωμένη βάση, την οποία μετά ο καθένας θα μπορεί να αξιοποιήσει εύκολα και γρήγορα.

Τα δεδομένα αφορούν την εξάπλωση του COVID-19 κυρίως στην Ελλάδα αλλά και σε παγκόσμια κλίμακα. Συνδυάζουμε δεδομένα από διάφορες πηγές και επίσημους φορείς όπως:

  • Ινστιτούτο Johns Hopkins CSSE
  • Εθνικός Οργανισμός Δημόσιας Υγείας (ΕΟΔΥ)
  • Παγκόσμιος Οργανισμός Υγείας (ΠΟΥ)

Τα δεδομένα που παρέχουμε ενημερώνονται αυτοματοποιημένα από τις πηγές τους ημερησίως για να είναι όσο το δυνατόν πραγματικού χρόνου.

2. Συντελεστες


  • Έρευνα & Υλοποίηση: Αλέξανδρος Δελητζάς, Φαίδων Βαγιωνάς, Βασίλης Ματσούκας

Λυση 2: Αναπτυξη API για την παροχη real-time δεδομενων

  1. 1. Περιγραφη
  2. 2. Συντελεστες
  3. 3. Συνδεσμοι

1. Περιγραφη


Τα APIs (Application Programming Interface) αποδείχθηκαν πολύ χρήσιμα εργαλεία για την παρακολούθηση της εξάπλωσης του COVID-19 σε παγκόσμια κλίμακα.

Ο στόχος μας είναι η ανάπτυξη ενός API με σκοπό την παροχή real-time δεδομένων εξάπλωσης του ιού στην Ελλάδα. Η υπηρεσία παροχής δεδομένων θα διατίθεται για δημόσια χρήση χωρίς χρέωση.

Το κίνητρο μας είναι να εξασφαλίσουμε έναν εύκολο και γρήγορο τρόπο να λάβει ένας ερευνητής/αναλυτής αξιόπιστα δεδομένα πραγματικού χρόνου, τα οποία έχουν προκύψει από συνδυασμό ποικίλων πηγών.

2. Συντελεστες


  • Έρευνα & Υλοποίηση: Αλέξανδρος Δελητζάς, Βασίλης Ματσούκας, Νίκος Παπαδημητρίου, Σταμάτης Αλειφέρης, Δημήτρης Μανωλίδης
  • Υποστήριξη: Ομάδα εθελοντών Project Data Analytics, COVID-19 Response Greece
